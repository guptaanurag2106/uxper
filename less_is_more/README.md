Implementing Text Classification

- Original Research Paper [Less Is More](https://arxiv.org/abs/2212.09410)
- Dataset used [AG Dataset](https://www.kaggle.com/datasets/amananandrai/ag-news-classification-dataset?select=test.csv)
- Compression library used [libdeflate](https://github.com/ebiggers/libdeflate)
- Run: `make && ./main ./AG_Dataset/train.csv AG_Dataset/test.csv`

Accuracy over the first 50 elements in AG_Dataset/test.csv: 79.59%
