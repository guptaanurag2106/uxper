# Deque

Implementation of the Deque concatenative programming language in Go.

## Context
- [Deque Language](https://www.concatenative.org/wiki/view/Deque)
- [Rule 110](https://en.wikipedia.org/wiki/Rule_110)

## Build & Run

```bash
# Build
go build

# Run (or run with a file)
./deque rule110.deque
```
