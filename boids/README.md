# Boids Flocking Simulation

A flocking simulation based on the Boids algorithm, implemented in C using SDL2.

## Context
- [Boids - Wikipedia](https://en.wikipedia.org/wiki/Boids)
- [Craig Reynolds' Boids](http://www.red3d.com/cwr/boids/)

## Build & Run

```bash
# Build (Requires SDL2)
make

# Run
./main
```
